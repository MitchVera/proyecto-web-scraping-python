# Instrucciones para ejecutar el proyecto

1. Instalar las dependencias:
   - Ejecutar el siguiente comando:
     pip install Flask beautifulsoup4 requests transformers torch sqlite3

2. Ejecutar la base de datos:
   
   - El archivo 'db.sqlite' es el archivo de base de datos.
   - En caso de que el archivo de datos no esté presente el comando 'python database.py' (Sin comillas) en el directorio del proyecto creará una nueva base de datos (Con las columnas     	respectivas).

3. Ejecutar el servidor Flask: 
   - Navegar al directorio del proyecto y ejecutar: 'python app.py' (Sin comillas)
   - El servidor estará disponible en http://127.0.0.1:5000

4. Usar los endpoints:
   - POST /scrape: Para extraer titulares y descripciones de una URL.
   - POST /process: Para procesar un texto y obtener el análisis de sentimiento.
   - POST /combined: Para combinar el scraping y procesamiento en un solo paso.


5. Pruebas:
   - Las pruebas se realizaron con Postman.

	Ejemplo:

	Solicitud POST a http://127.0.0.1:5000/scrape

	Solicitud POST a http://127.0.0.1:5000/combined
